This project was implemented as part of the Intelligent Systems Techniques module of
the MSc Artificial Intelligence & Adaptive Systems at Sussex University. This project 
got the highest grade among all the submissions for the year 2019-2020.
In order to run the program you can either open it and run it from Netbeans 
or just open the IST_Checkers Executable Jar File within the dist directory.

The provided code has been tested on Windows 10.
